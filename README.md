# Estructuras de Control y Arreglos en PHP

Autores: Abner Moises Sarmiento Valencia, Carlos Roberto Silva Chacón, Miriam Vázquez Guzmán

Mini sitio web de ejemplo que muestra **condicionales** (`if`, `elseif`, `else`, `switch`), **bucles** (`for`, `while`, `foreach`) y **arreglos** (indexados, asociativos, multidimensionales) en PHP.

## Estructura
```
/ (raíz)
├── index.php
├── condicionales.php
├── bucles.php
├── arreglos.php
├── css/
│   └── styles.css
├── includes/
│   ├── header.php
│   ├── menu.php
│   └── footer.php
└── img/
```

## Requisitos cubiertos
- 4 páginas PHP conectadas con `<nav>` (reutilizado con `include()`).
- CSS externo (`/css/styles.css`).
- Variables, `echo`, `date()`, `count()`.
- Condicionales (`if/elseif/else`, `switch`), bucles (`for/while/foreach`).
- Arreglos indexados, asociativos y multidimensionales, recorridos con control de flujo.
- Comentarios breves en el código.

## Cómo ejecutar localmente
1. Instala PHP (8.x recomendado).
2. En la carpeta del proyecto, abre una terminal y ejecuta:
   ```bash
   php -S localhost:8000
   ```
3. Abre <http://localhost:8000> en tu navegador.
